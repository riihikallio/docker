# Perl-ympäristö harjoitusta 1.17 varten

Ubuntu-pohjainen Perl-ympäristö

## Asennus

```bash
docker build -t perl riihikallio/perl
```
Nykyisen hakemiston .pl-päätteiset tiedostot kopioidaan konttiin.

## Käyttö

```bash
docker run perl skripti.pl
```
